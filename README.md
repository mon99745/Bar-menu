# Bar-menu
현 가게들에서 사용되는 태블릿 메뉴판에 들어갈 수 있는 웹 어플리케이션입니다.

  - Spring Security 를 이용한 유저서비스
  - JPA와 MySQL 을 이용한 RDBMS와 주문서비스
